Overview

Wellmily is a social wellness app that helps close groups manage and support health goals. This privacy policy explains what information we collect and how we use it.

Information we collect

  • Account information: email and display name for user accounts.
  • Usage data: anonymized usage events to improve the app (e.g., session counts, feature use).
  • Device permissions: We request access to the device camera only when the user chooses to use camera features (e.g., take a photo). Camera images are not shared externally unless the user explicitly chooses to share them.

How we use information

We use collected information to operate and improve the app, provide features, troubleshoot issues, and send important account or security messages. We do not sell personal data.

Third-party services

We may use third-party services for analytics and crash reporting (for example Firebase or Sentry). These services may process data on our behalf; we limit what they receive to anonymized or necessary data only.

Data retention & security

We retain account data as long as the account exists. We implement standard technical and organizational measures to protect data.

User choices

Users can delete their account and data via the app. Contact us at privacy@wellmily.com to request data removal or for privacy questions.
